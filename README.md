# Customer-Order-Revenue-Analytics-Dashboard

The Customer Order & Revenue Analytics Dashboard is a data visualization project built using Microsoft Power BI. The dashboard provides insights into customer purchasing behavior, product category performance, and regional order distribution.

This project focuses on transforming raw sales data into interactive and easy-to-understand visualizations, enabling businesses to identify trends, understand their customers, and make informed decisions.

The dashboard includes multiple analytical views such as age group analysis, gender distribution, occupation-based orders, state-wise order distribution, and top product categories.

By analyzing these metrics, businesses can better understand who their customers are, what they buy, and where the demand is highest.

Key Objectives:

*Analyze customer order patterns

*Identify top-performing product categories

*Understand customer demographics

*Track revenue and order distribution

*Support data-driven business decisions
